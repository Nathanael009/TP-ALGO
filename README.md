# TP-ALGO
Ce projet de Hackathon pour l'ISPM est un excellent défi qui combine la logique des jeux de société traditionnels avec l'algorithmique avancée. Le Fanoron-telo (très similaire au Tic-Tac-Toe dans ses conditions de victoire, mais avec une phase de déplacement) est parfait pour implémenter l'algorithme Minimax avec élagage Alpha-Beta.
